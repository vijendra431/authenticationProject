# 🛍️ React Authentication & Protected Routes App

## 🚀 Overview

This project is a React-based web application that implements **Authentication and Authorization** using JWT tokens. It demonstrates how to protect routes, manage user sessions, and securely fetch data from APIs.

---

## 🔧 Tech Stack

* React.js
* React Router (Protected Routes)
* JavaScript (ES6+)
* CSS
* js-cookie (for JWT storage)

---

## 🔐 Features

* User Login Authentication (JWT based)
* Protected Routes (Home, Products, Cart)
* Secure API calls with Authorization headers
* Logout functionality (clears JWT token)
* Redirect handling for authenticated/unauthenticated users
* Error handling for invalid login

---

## 🧠 Key Concepts Implemented

* Authentication vs Authorization
* JWT Token Handling using Cookies
* Route Protection using custom ProtectedRoute
* Navigation using history & Redirect
* Conditional Rendering based on authentication state
